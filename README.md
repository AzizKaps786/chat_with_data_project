# Chat With Your Data 📊

A Streamlit app that lets you upload CSV/Excel/JSON files and ask natural language questions. The Groq LLM generates pandas + Plotly analysis code, the app executes it, and returns a chart plus a plain-English insight.

## Run in Visual Studio Code

```bash
python -m venv .venv
# Windows PowerShell
.venv\Scripts\Activate.ps1
# macOS/Linux
# source .venv/bin/activate

pip install -r requirements.txt
copy .env.example .env   # Windows
# cp .env.example .env   # macOS/Linux
# Edit .env and add your GROQ_API_KEY

streamlit run app.py
```

Or press **F5** in VS Code using the included `.vscode/launch.json`.

## Safety note

The app executes code generated by an LLM. This implementation restricts imports and builtins, but for production you should run execution in a stronger sandbox/container.
